# Tournament Runner

## Overview

Ever want to run a gaming or sports tournament but hated doing pairings and rankings manually? This app aims to fix that pain point!

## Requirements

1. Create a tournament
2. Add participants to the tournament
3. Auto-select number of rounds based on number of participants
4. Decide matches based on ranking
5. Display rankings
6. Report match outcomes
7. Close tournament when completed

## Running the App

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
